# App Backend

Node.js based backend Template

## Install dependencies :

```    
$ npm install
```

## Run the app :

```     
$ npm start
```

## Docker Containers for Mongo & Postgres 

```bash 
$ docker-compose up -d

// Stop Containers
$ docker-compose down
$ docker stop $(docker ps -aq)

// Remove Containers
$ docker rm -f  $(docker ps -aq)
```
